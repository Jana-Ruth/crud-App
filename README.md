## Features

- Add new products (title, price, description, image)
- Edit existing products using a modal form
- Delete products with success messages
- Live image preview while adding/editing products
- Responsive design with Tailwind CSS
- Fully typed with TypeScript

## Technologies

- React.js (Functional components & Hooks)
- TypeScript
- Tailwind CSS
- Vite
- FakeStore API for product data
